themeのインストール手順

hexoのテーマを使うには以下の流れでインストールして、テーマを有効にします。

  1. 使いたいテーマのファイルをHexoやgithubで探してダウンロード
  2. 自分のhexoのプロジェクト直下にあるthemesディレクトリに移動
  3. themeディレクトリに手順1のファイルをコピー
  4. 自分のhexoのプロジェクト直下にある_config.xmlを開く
  5. _config.xmlの中のtheme: ***を手順1のディレクトリ名に変更

主なhexoのテーマは以下から探す事ができます。

Hexo公式サイト テーマ一覧
Hexo - themes

githubではHexo公式サイトに登録されていないテーマを探すことができます。

おすすめのtheme一覧

それでは早速、Hexoのテーマをご紹介したいと思います。デモページのスクリーンショット画像は、左から幅320px時の表示と1024px時の表示をまとめています。

Icarus

Icarus ©

段組が美しくて安定感のあるデザインです。写真ブログやアイキャッチの画像をはめ込むような、イメージ・コンテンツを見せたいブログで目映えが良さそうです。各メニューへのアクセスやディプレイのサイズに応じてレスポンシブに変化しくなど、隙のない作りとなっていますので、インストールして速攻で使えるテーマです。

hexo-theme-icarus
github - hexo-theme-icarus
テーマのDEMO - プレビュー

Tranquilpeak

Tranquilpeak ©

こちらも段組が美しくて安定感のあるデザインです。Icarusに比べてサムネイルなどが控えめとなっており、静かで落ち着いた雰囲気を求めるならこちらが良さそうです。最近、日本人の方が立ち上げたブログを拝見しますと、このテーマを使っているケースをよく見かけるようになりました。

hexo-theme-icarus
github - hexo-theme-tranquilpeak
テーマのDEMO - プレビュー

hexo-theme-hiker

見た目がシンプルでビジュアルと伴にすっきり魅せる事ができるテンプレートです。記事の目次や、メニューのカテゴリーなども見やすくとても使いやすそうな印象です。

hexo-theme-hiker
github - hexo-theme-hiker
テーマのDEMO - プレビュー

hexo-theme-apollo

Apollo ©

日本語の記事がすっきりと表示されるクールなテーマで、技術系ブログなど、長文を疲れさせることなく読ませるサイトに向いているかと思います。面白いことに、このテーマはSANOGRAPHIX.NETさんがTumblr向けに制作されているapollo themeを、@joyceimさんがHexoに移植したテーマだそうです。こちらのブログでもhexo-theme-apolloをオレオレ仕様でカスタマイズして使っています。

hexo-theme-apollo
github - hexo-theme-apollo
テーマのDEMO - プレビュー

hexo-theme-snow

こちらは日本の方が制作されているテーマです。制作者である@natsuhikokさんのサイトを拝見しますと、特に日本語やコードブロックがとても読みやすい設計になっていて、hexo-theme-apolloと同様、じっくり読ませる記事に向いているかと思います。日本の方がシンプルなデザインからHexoブログを始めたい時には、心強いテーマの一つだと感じました。

hexo-theme-snow
github - hexo-theme-snow
テーマのDEMO - プレビュー

hexo-theme-ghost-casper

シンプルなブログ・プラットフォームとして人気を博しているGhostのデフォルト・テーマ「casper」を@MIKAGMRさんがHexoに移植したものです。デザインはシンプルさとオシャレさを兼ね備えています。また見た目のシンプルさによらず機能は充実しているようです。

hexo-theme-ghost-casper
github - hexo-theme-ghost-casper
テーマのDEMO - プレビュー

Light

Light ©

こちらはHexo純正のシンプルテーマ。Wordpressで言うところの「 Twenty 〜」的な立ち位置でしょうか。まずは無難なデザインからブログを始めたい方はこちらがオススメです。hexoをインストールすると標準で使えるlandscapeと似たデザインとなっていますが、Lightはlandscapeをよりシンプルにした白基調のデザインです。

hexo-theme-light
github - hexo-theme-light
